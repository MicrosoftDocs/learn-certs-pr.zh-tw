# 關於在線考試

您可以在家中或辦公室舒適地參加任何基於角色的在線考試或基礎考試，同時由監考人員通過網絡攝像頭和麥克風進行監考。（通過 K-12 學校購買的基礎知識和 Microsoft 技術助理（MTA）考試無法在線進行。）如果您有興趣參加線上測驗，而且在註冊之前想要更好地理解可能出現的情形，則此頁面將會解釋詳細資訊。您也可以觀看英語或带阿拉伯語，法語，德語，葡萄牙語或西班牙語字幕的[YouTube視頻](https://www.youtube.com/playlist?list=PLahhVEj9XNTceGnF19qSjYxYzS4S_Wlmf) 。


## 進行系統測試

現在參加系統測試，檢測您的電腦和地址是否和在線測驗軟體相容，然後安排您的考試時間。

[立即進行系統測試](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)

此頁面列出目前有提供線上測驗的國家。注意，接待員支援、監考者支援和監考軟體目前只提供英文版。

**警告！在註冊在線考試之前：**

1. **在要進行測試的同一台計算機以及同一位置上，必须完成[系統測試](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)。**  
2. **確認您的ID符合要求（請閱讀[要求](https://home.pearsonvue.com/Policies/1S/English)），並且您的個人資料中的姓名與您的身份完全匹配。**  
3. **查看下面的技術，系統，環境和測試協議要求。**

## 系統要求

線上測驗可能受 Proxy 伺服器、網路封包檢查/篩選或嚴格網路安全性組態干擾。

登入交付工作站所用的 Windows 使用者帳戶，必須具有本機系統管理的權限。

部分防毒軟體可能造成效能問題。進行測驗時，請停用防毒掃描。

您可能需要確保您的安全性軟體不會封鎖 JavaScript、Adobe Air 應用程式或 Adobe Flash Player 擴充套件的執行。

您可能需要設定網路安全性及防毒軟體，以使 Pearson Secure Browser 可執行檔案成為例外。

## 參加所在國/地区

如果在您的國家有提供線上測驗，測驗即能夠以您國家的測驗中心所能提供的所有語言來進行。請注意，接待員支援、監考者支援和監考軟體目前只提供英文版。在日本以外的大多數國家都可以進行線上考試。

## 測試期間協議

為維護線上/網路攝影機設定中的監考測驗誠信，將會執行特定原則、做法和程序。

### 辦公室或家庭環境

您的辦公室或住家設定必須滿足下列要求：

- 辦公室或會議室必須四面有牆，大門緊閉且不會受干擾。
- 如果要在家中參加考試，則房間應無干擾。
- 在測試期間，不得有第三方在場。

與接待員或監考者連絡：  

- **電話。** 在簽到程序中，接待員會在您測驗簽到時請您提供與您連絡的電話號碼。在完成簽到程序之後，會要求您將電話放在拿不到但又近到可以聽到的地方。
- **聊天窗口。** 與接待員和監考者的連絡大多會經由交談視窗進行。
- **麥克風。** 在簽到程序及測驗 (若有需要) 期間，您可以經由麥克風與接待員和監考者連絡。
- **語言。** 與接待員和監考者之間的所有連絡會均以英語進行。

身份驗證：

您必須出示現行有效並由政府核發的證件，例如下列其中一：

- 駕照
- 軍人證
- 護照
- 身份證（國家/州/省身份證）
- 外國人登記證（綠卡，永久居民，簽證）
- 政府簽發的本地語言ID（非羅馬字符）——_僅當ID由應試者所在的國家/地區發出時才被接受_

網路攝影機將會拍攝您的證件和您的相片。

證件上的姓名必須符合您 Microsoft 和 Pearson VUE 個人檔案中的姓名。

18 歲以下青少年可出示有效學生證作為身份證明。應試者的監護人也必須出示有效證件，並且必須在簽到程序期間口頭同意測驗。

工作區域掃描：

您必須使用網路攝影機，或抬起並旋轉筆記型電腦，從左到右緩慢掃瞄四周牆壁，並且上下掃瞄以顯示地板和天花板。如果無法完成房間掃瞄，則不得繼續進行或參加測驗。

接下來您必須掃描整個工作區域。監考者會檢視安全性風險，包括但不限於

- 其它的顯示器（必須拔下電源，並遠離您）
- 其它的計算機（必須關閉並且顯示器必須為黑屏）
- 壁畫/海報（將接受檢查）

工作區域應該清空所有資料，包括下列物品 (不得放在手拿得到的地方)：書籍、筆記本、便利貼筆記、打字筆記 / 紙張或書寫工具 (例如筆、馬克筆或鉛筆)。白板應該擦乾淨。

### 沒有個人物品

- 在考試期間，您不能接觸以下類型的個人物品：手提電腦或其他電子設備，傳呼機，手錶，錢包，皮夾，帽子（或其他頭上用品），包，大衣，書籍，便箋或任何其他未明確批准的材料。
- 要求您把所有的口袋（褲子的口袋和夾克/毛衣的口袋）全部翻出來，以便接待人員看到它們是空的。
- 如果您穿著長袖，請捲起袖子以表明手臂上沒有任何文字。
- 如果您系領帶，則要求提起領帶以向接待人員表明領帶下沒有藏任何東西。
- 如果長發遮住耳朵，則要求您將頭髮向後撈起，以向接待人員表明耳朵上沒有戴任何東西（例如藍牙耳機）。

### 不得休息/飲食

- 無論出於任何原因，考試期間都不得休息。您理解並且同意：如果中斷，測驗分數無效且之後不得繼續進行測驗。
- 考試期間禁止飲食，吸煙和嚼口香糖。

### 沒有考試協助

- **在測試期間，任何第三方不得進入房間。如果發生這種情況，您的考試將被終止。**
- 參加考試時，您不能得到其他人的幫助或指導，並且您不能允許其他人看到呈現考題的計算機屏幕。
- 在考試期間，您不得使用考試專用材料，也不得使用草稿紙，可擦白板或任何其他書寫材料。測驗交付期間也不得使用任何書寫工具，例如筆、馬克筆或鉛筆。
- 監考人不能回答與考試內容有關的問題。如果有關於測驗內容的問題，請在完成測驗之後連絡 Microsoft。

### 其他重要信息

- 您不得以任何方式複製或記錄考試中的考題或答案，也不得與他人分享或討論考試中出現的考題或答案。
- 參加此次考試，即表示您承認監考人員會在您參加考試時不斷通過視頻和音頻監視您，而且承認並同意在考試交付期間對您的面部、聲音、就座空間以及位置進行視頻和音頻錄製。您認可在測驗期間於目前位置沒有隱私權，並且您放棄主張隱私權的任何所有權利。如果您不想以視訊或音訊記錄您，請立即告知監考者您不要繼續進行測驗的意圖。
- 如果您不遵守[應試者規則協議](https://www.pearsonvue.com/rp/rp_candidate_rules_agreement.pdf) 或者懷疑你作弊或篡改了 Pearson VUE 或其分包商的軟件，或者是如果在考試期間在您所在的物理房間中檢測到第三方，則監考人將立即終止考試並將違規情況連同音頻和/或視頻證據報告給 Pearson VUE 和 Microsoft。您的測驗可能會無效，並且 Microsoft 可能會採取其他行動，例如取消您的資格並在未來不准您參加所有 Microsoft 認證計劃。

## 啟動考試和應試者報到過程

啟動和簽到程序應該會花費約 15 分鐘。

- 關閉瀏覽器彈出窗口阻止程序。
- 確保您具有計算機的管理權限，以便可以下載測試軟件。

若要開始測驗階段，並與接待員連線：

- 登錄至[您的福利和考試儀錶板](https://www.microsoft.com/learning/dashboard.aspx)。
- 在右側的"預約"部分中，選擇"開始先前安排的在線監考考試"。
- 將出現您安排的考試。選擇檢查鏈接，然後選擇"開始考試"。

**注意**您最多可能需要等待五分鐘，才能與接待人員聯繫上。

與接待員連線之後：

- 提供駕照，護照或其他可接受的身份證明，以便驗證您的身份。
- 接待人員將在考試交付期間審查考試行為，測試環境和溝通過程。與接待員和監考者之間的所有連絡會用英語進行。
- 將要求您關閉所有非必需的應用程式（例如電子郵件或聊天）。

在此程序完成之後，測驗即會開始。除非有問題，否則監考者在測驗期間不會連絡您。

如果在測驗交付期間發生技術問題，監試者會先嘗試經由Chat連絡您。如果不成功，監試者會用電話連絡您。

如果在測驗交付之後還有問題，請連絡 Pearson VUE。請提交監考者所提供的事件報告編號。

### 獲取成績報告，並刪除監考人軟件

可以從[您的福利和考試儀錶板](https://www.microsoft.com/learning/dashboard.aspx)訪問成績報告。

- 通過[www.microsoft.com/learning](https://www.microsoft.com/learning)登錄。
- 要查看您的考試歷史記錄，請轉到[您的福利和考試儀表板](https://www.microsoft.com/learning/dashboard.aspx)，然後選擇"查看我的成績報告"。您可以從這裡儲存和列印成績單。
- 您的考試結果（通過或失敗）將轉發給 Microsoft 進行處理。
- 考試結束後，您應該刪除所有可執行文件和輔助文件，包括"Pearson VUE 安全瀏覽器"圖標。要從桌面上刪除該圖標，請右鍵單擊該圖標，選擇"刪除"，然後選擇"是"。
- **[卸載 Pearson VUE 安全瀏覽器的可執行文件](https://windows.microsoft.com/zh-cn/windows7/uninstall-or-change-a-program)**。

## 註冊在線考試

在線考試適用於在參加所在國家/地區進行的基於角色的考試或基礎知識考試。如果在您的國家有提供線上測驗，測驗即能夠以您國家的測驗中心所能提供的所有語言來進行。請注意，接待員支援、監考者支援和監考軟體目前只提供英文版。通過 K-12 學校購買的 Microsoft Office 專家（MOS），基礎知識和 MTA 考試無法在線進行。要註冊，請從[考試列表](https://www.microsoft.com/learning/exam—list.aspx)中選擇您要參加的 Microsoft 在線考試，然後選擇"安排考試"按鈕。如果在您的國家有提供線上測驗，您將可以在測驗註冊過程中選取該選項。

## 常見問題集

尋找線上監考測驗相關常見問題的解答。

### 我如何才能找到有關在線考試的更多信息？

Microsoft 發佈一個短片，它們會告訴您參加測驗是多麼簡單。該視頻可以在[YouTube播放列表](https://www.youtube.com/playlist?list=PLahhVEj9XNTceGnF19qSjYxYzS4S_Wlmf)中觀看，視頻為英語或帶有阿拉伯語，法語，德語，葡萄牙語和西班牙語字幕。

### 我怎麼知道在線考試是否適合我？

首先，必须完成[系統測試](https://service.proctorcam.com/system_test?customer=pearson_vue&clientcode=MICROSOFT)，確保您的計算機滿足我們的要求。通過[測試期間協議](https://www.microsoft.com/learning/online—proctored—exams.aspx#protocol—label)了解可能會發生的事項。如果您有任何疑問或保留，建議將測驗排定在測驗中心。

### 如果我的系統未通過系統測試，我仍可以在線參加考試嗎？

不行。如果系統沒有通過系統測試，包括確保您可以拍攝清楚的證件相片，則不得註冊線上測驗。

### 如何註冊在線考試？

要註冊，請從[考試列表](https://www.microsoft.com/learning/exam—list.aspx)中選擇您要參加的考試，然後選擇"安排考試"按鈕。如果在您的國家有提供線上測驗，您將可以在測驗註冊過程中選取該選項。

### 如果我不在考試中心，作弊似乎會容易得多。線上測驗是否安全？

我們致力於保護測驗的誠信，無論測驗用什麼方式交付。當您參加線上測驗時，必須先安裝安全瀏覽器，以在整個測驗過程中執行強力的安全保護措施。此外，Pearson VUE 監考者在測驗期間會經由網路攝影機和麥克風來全時監看您。當我們懷疑有作弊時也會檢查測驗期間的錄影片段。我們很擅長抓作弊，所以不建議您嘗試。我們會取消作弊者的資格、移除作弊者的認證並禁止參加未來測驗。（[閱讀我們的其中一個反盜版 PM Kerri Davis 所給的其它觀點](https://www.microsoft.com/zh—cn/learning/community.aspx)。）

### 如果我要參加多項考試以獲得一个認證，是否需要選擇一種考試交付方式？

否，交付方法可以混搭（線上或親臨測試中心），測驗內容都是相同的。

### 考試已啟動並分配了監考人後，我需要多長時間才能完成考試？

可用於完成測驗的時間將會與您在 Pearson VUE 測驗中心參加測驗時相同。

### 在線考試比考試中心的考試更難嗎？

不會。測驗內容都是相同的。

### 我可以在在線考試時做筆記嗎？

是。白板功能提供輸入筆記或繪製形狀的選項。您可在測驗期間單擊熒屏上方的橡皮擦圖標存取白板。

### 在線考試是否支持多種語言？

是。如果在您的國家有提供線上測驗，測驗即能夠以您國家的測驗中心所能提供的所有語言來進行。請注意，監考軟體、接待員支援和監考者支援目前只提供英文版。

### 如果在在線考試之前、之中或之後遇到技術問題，我應該與誰聯繫？

關於註冊問題，請使用註冊期間提供的 Pearson VUE 客戶服務連結。如果您在測驗交付期間遭遇技術問題，測驗監考者會協助您嘗試解決問題。如有必要，在跟進[Pearson VUE 客戶支持](https://www.pearsonvue.com/microsoft/contact)時，監考人員將提供事件報告編號以供參考。

### 辦理登記手續時需要什麼類型的身份證明？

請查看當前的[ID政策](https://home.pearsonvue.com/Policies/1S/English)。如果您正在更新或換新證件，請注意線上測驗不接受法定名稱變更表格，包括身份證換新表格。

### 如何更改我的法定姓名？

您的名稱必須完全符合您在簽到程序期間提供的證件。保存考試註冊資料後，您只能通過與[Microsoft 區域服務中心](https://www.microsoft.com/learning/help)聯繫來更改您的法定姓名。

### 我為在線考試安排了預約。要如何開始？

在排定的預約時間之後，您最多有 15 分鐘來開始測驗。在指定的 15 分鐘之後即不得開始測驗。

要開始考試，請登錄[福利和考試儀錶板](https://www.microsoft.com/learning/dashboard.aspx)，然後單擊"開始之前安排的在線監考考試"。您將會重新導向到 Pearson VUE 頁面，在此頁面您會看到所排定的測驗，然後就可以開始測驗。

### 在哪裡可以找到考試政策列表？

請參閱適用於所有 Microsoft 認證考試的[考試政策和常見問題解答](https://www.microsoft.com/learning/certification-exam-policies.aspx)。
