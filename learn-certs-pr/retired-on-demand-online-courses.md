# Microsoft 隨選課程已淘汰

截至2020年6月30日，所有Microsoft官方課程(MOC）按需課程(包括以下所列課程）均已停用。  對於在停用日期之前購買或獲得MOC按需課程代碼的客戶，您有90天的時間激活課程，然後在激活後的90天以完成課程，此後將不再提供課程。

請在以下站點上參考免費的在線課程和學習資源：[https://aka.ms/learn](https://aka.ms/learn)
