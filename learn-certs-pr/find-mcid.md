# 查找您的 MCID

如需查找您的 Microsoft Certification identification number (MCID)，請登錄您的 [Certification Dashboard。](https://aka.ms/certdashboard)您的 MCID在該頁面的右上角。

![MCID 在該頁面的右上](images/certification-dashboard-find-mcid.png)
